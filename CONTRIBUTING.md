# Contributing

This is currently Selva's personal collection of agent skills for startup engineering. Contributions of new skills or substantial changes to existing skills are **not open at this time**.

This isn't because outside input isn't welcome — it's because this collection reflects a specific voice and opinions that Selva wants to maintain consistently. Opening the collection to contributions would dilute that coherence.

## What you can do

- **Open an issue** to report bugs, suggest improvements, or ask questions. Thoughtful feedback is welcomed.
- **Fork this repo** to create your own variant with your modifications. This is entirely fine under the MIT license.
- **Build your own skills repo** and share it with the community. The agentic engineering ecosystem benefits from more authored skills, not fewer.

## Reporting issues

If you find a skill that produces unexpected or incorrect results:

1. Note the skill name and what happened
2. Include the context (what task you were working on)
3. Open an issue with this information

Issues are reviewed when Selva has time — not immediately, but consistently.

## Thanks

If these skills are useful to your team, a star on the repo is appreciated.
