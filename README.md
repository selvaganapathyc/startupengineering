# Selva's Startup Engineering Skills

Agent skills for startup engineering — authored by Selva Ganapathy.

These skills encode my thinking on how to build products well within
startup constraints. Each skill is a standalone artifact your agent
can consume directly. For context, curation, and decision guidance
about when to reach for which skill, visit startupengineering.io.

## Install

Install the full collection:

    /plugin install selvaganapathyc/startupengineering

Individual skills are also installable — see each skill's folder.
